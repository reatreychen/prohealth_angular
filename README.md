# ProHealth Frontend (Angular)

A premium hospital management frontend built with Angular, featuring a modern design and dynamic data integration.

## Features
- **Modern UI**: Polished, responsive design using Tailwind CSS.
- **Dynamic Content**: Live data from NestJS API (Doctors, Departments, Blog, Shop).
- **Admin Tools**: Integrated management for medical departments and content.
- **Form Handling**: Reactive forms with validation and file upload support.

## Getting Started
1. Install dependencies: `npm install`
2. Ensure Backend is running at `http://localhost:8080`.
3. Start server: `npm run start` (or `ng serve`)
4. Open: `http://localhost:4200`
